

# **Формат входного файла(txt)**

(Source Target Weight)

0 1 10

0 2 15

1 2 20

...

# **Алгоритм**

**Принцип работы**

1.  **Инициализация**: Создание матрицы феромонов
2.  **Построение путей**: Каждый муравей строит путь по графу
3.  **Оценка путей**: Вычисление длины каждого пути
4.  **Обновление феромонов**: Усиление феромонов на лучших путях
5.  **Испарение**: Постепенное уменьшение уровня феромонов
6.  **Повторение**: Цикл повторяется заданное количество итераций

**Параметры алгоритма**

- ants - количество муравьев (по умолчанию: 20)
- iterations - количество итераций (по умолчанию: 100)
- decay - коэффициент испарения феромонов (по умолчанию: 0.1)-
- alpha - влияние феромонов на выбор пути (по умолчанию: 1.0)
- beta - влияние расстояния на выбор пути (по умолчанию: 2.0)

# **Основные функции**

**read_graph(filename)**

Читает граф из файла и создает матрицу расстояний.

**Логика работы:**

- Чтение ребер из файла
- Определение размера графа
- Создание матрицы расстояний
- Заполнение матрицы весами ребер

**ant_algorithm(distances, ants, iterations, decay)**

Реализует основной алгоритм муравьиной колонии.

**Логика работы:**

- Инициализация матрицы феромонов

  
  Для каждой итерации:
- Каждый муравей строит путь
- Вычисляется длина пути
- Обновляется лучший путь
- Происходит испарение феромонов
- Обновляются феромоны на лучшем пути

**multiple_runs(filename, num_runs, ants, iterations)**

Запускает алгоритм несколько раз для поиска наилучшего решения.

# **Результаты**

**Выходные данные**

Программа выводит:

- Найденный оптимальный путь
- Длину оптимального пути
- Графики зависимости феромонов и длины пути от итераций

**Графики анализа**

1.  **Феромоны от итераций** - показывает динамику изменения уровня феромонов
2.  **Длина пути от итераций** - демонстрирует улучшение качества решения
